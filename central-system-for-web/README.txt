  
instalación de la aplicación

1- Ejecutar XAMPP
2- Crear una carpeta de proyecto en Htdocs
3- Ejecutar el comando
	 git clone https://github.com/RoxyAristegui/central-system-api
4- Cargar el archivo central-system-api.sql en phpmyadmin

	
5- funcionamiento de la interfaz:
acceder a localhost/proyecto/central-system-for-web
se ejecutará el archivo index, introducir los datos de una organización para visualizarlo en la lista.
El boton rojo permite eliminar un regitro, y el botón Ver, visualizarlo y editarlo.
