instalación de la aplicacion:

1- Ejecutar XAMPP
2- Crear una carpeta en Htdocs
3- ejecutar el comando:
	$ git clone https://github.com/RoxyAristegui/central-system-api
4- Cargar el archivo central-sistemapi.sql en phpmyadmin
5- Ejecutar las llamadas en 
proyecto/central-system-api/Organizations.php

las peticiones aceptadas:
-GET : con y sin parámetros, con el parametro ?id se accede a las organizaciones deseadas.
-POST
-PUT
-DELETE
